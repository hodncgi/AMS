# Figma design preview (Week 2)

Static HTML mockup based on the **manager-provided Figma template**.

## Open locally

Double-click or:

```powershell
start frontend/design-preview/index.html
```

No npm required. This file is **not** connected to the API (layout reference only).

## Live chat (Week 4)

Start the backend, then use the working UI:

**http://localhost:8000/ui/**

See [../chat-live/README.md](../chat-live/README.md).

## What this demonstrates

- CGI branding and tab structure from Figma  
- Planned **Onboarding** vs demo tabs  
- Placeholder chat layout with source citations (US-13 preview)  

## What this is not

- Not the live chat (use `/ui/` on port 8000)  
- Not the final React app (Week 8)

See [../../doc/design/FIGMA_TEMPLATE_REVIEW.md](../../doc/design/FIGMA_TEMPLATE_REVIEW.md).
