# Figma design preview (Week 2)

Static HTML mockup based on the **manager-provided Figma template**.

## Open locally

Double-click or:

```powershell
start frontend/design-preview/index.html
```

No npm, no backend required.

## What this demonstrates

- CGI branding and tab structure from Figma  
- Planned **Onboarding** vs demo tabs  
- Placeholder chat layout with source citations (US-13 preview)  

## What this is not

- Not connected to FastAPI or LM Studio (Weeks 8–9)  
- Not the final React app (Week 9)
